# 1.说明

```bash
.
├── CMakeLists.txt
├── example
│   └── CMakeLists.txt
├── include
│   ├── hello.hpp
│   └── libs_add
│       └── add.hpp
├── lib
│   ├── add.cpp
│   └── CMakeLists.txt
├── src
│   ├── CMakeLists.txt
│   └── main.cpp
└── test
    ├── CMakeLists.txt
    ├── sub_foo
    │   └── sub_test_1.cpp
    ├── test_1.cpp
    └── test_2.cpp
```

- `example`：代码示例存放处；
- `include`：头文件存放处；
- `lib`：动态库/静态库存放处；
- `src`：源文件存放处；
- `test`：测试案例存放处。
